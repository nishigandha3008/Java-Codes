## 📖 About

This repository contains Java programs created while learning Java programming. It covers basic syntax, control statements, loops, arrays, methods, strings, object-oriented programming concepts, exception handling, and file handling.

The repository is designed for beginners who want to strengthen their Java fundamentals through hands-on coding practice.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Basic-Java-Programs.git
```

### Open the project

Open the project in:

- VS Code
- IntelliJ IDEA
- Eclipse

### Compile

```bash
javac FileName.java
```

### Run

```bash
java FileName
```
